Azure API Management for FHIR Server

Deploy Azure API Management for any FHIR server

This service provides a utility for generating a Swagger API defintion for a given FHIR endpoint.

You can test this Swagger generation service by entering a FHIR server URL below.

To deploy a API Management instance for a given FHIR endpoint to Azure, use the deploy buttons below. If you would like OAuth authentication enabled on your API, you will need appropriate Azure Active Directory application registrations. You can find instructions on how to create application in the Azure documentation. There are also instructions on how to configure Azure Active Directory applications and deploying API Management in the source code repository for this service: https://github.com/hansenms/fhir2azureapim
